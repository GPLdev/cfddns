GNU Bash Script

 

Simple bash script and intuitive to use crafted for updating the public dynamic IP address to an existing Cloudflare DNS record. The script is designed to avoids unnecessary traffic and optimizes API requests by tracking data through logs. To monitor any change of IP address, the script checks through DNS, STUN, and HTTPS. Simplify your system management by using simple scripts for straightforward tasks. Don't try to overcomplicate things by using complex software for simple API calls.
